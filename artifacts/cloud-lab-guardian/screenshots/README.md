# Screenshots

Add screenshots here before pushing to GitHub.

## Required screenshots

| Filename | What to capture |
|---|---|
| `home.png` | The landing page with the prompt input form visible |
| `overview.png` | A generated plan — the Overview / detected service cards section |
| `security.png` | The Security tab open, showing the always-on safety warnings |
| `cost.png` | The Cost tab open, showing free-tier vs paid-risk breakdown |
| `cleanup.png` | The Cleanup tab open, showing teardown checklist and commands |
| `readme-export.png` | The exported README.md open in a text editor or GitHub preview |

## How to take them

1. Run the app locally or open the Replit preview
2. Submit one of the golden prompts (e.g. "Build a serverless image uploader.")
3. Screenshot each tab at 1280×800 or wider
4. Save as PNG with the exact filenames above

## Tips

- Use a $0 budget plan for the security/cost/cleanup screenshots (cleanest output)
- Use "Build an app with EC2, RDS, NAT Gateway, and public S3." for the high-risk cost screenshot
- Use "Build a Lambda Function URL API." for the cleanup screenshot to show Function URL teardown
